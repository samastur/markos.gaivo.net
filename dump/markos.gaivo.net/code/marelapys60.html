Marela uploader for series 60 phones

Marela uploader for series 60 mobile phones with Python installed. You can either snap a photo or select one already taken, add tags, set privacy and upload it to Marela.

Demonstration

None for now. This is the first release and should be avoided. Only for the brave and self-sufficient. Last frontier, that sort of a thing.

This program doesn't eat kids, but it may eat your data.

Usage

Interface should be pretty self-explanatory. If it isn't, please let me know and I'll try to get around writing proper documentation. I sadly don't have time to do so without a good reason.

Requirements

A series 60 phone with 2nd Edition Python S60 1.2 installed and a lot of guts. Also working knowledge of slovene unless you turn off translation in code (which you are free to do).

Downloads

You can download Python script that should work on any aforementioned phone or a standalone application package tested on Nokia N70 (Python needs to be installed separately).

Bugs & limitations

  • supports only Marela service
  • file browser should only list images and directories
